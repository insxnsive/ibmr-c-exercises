## 🎓 Sobre o Projeto

Este repositório foi criado **por alunos, para alunos** do curso de Tecnologia da Informação do **IBMR** — e está aberto a estudantes de qualquer outra universidade que queiram aprender ou praticar a linguagem **C**.

Aqui você encontrará exercícios e exemplos organizados por tema, cobrindo desde o básico até tópicos mais avançados da linguagem. O objetivo é servir como material de apoio para quem está começando, bem como referência rápida para quem já tem alguma experiência.

---

## 📂 O que você vai encontrar

| Tópico | Descrição |
|---|---|
| **Hello World** | Primeiro programa, estrutura básica de um código em C |
| **Variáveis e Tipos** | `int`, `float`, `double`, `char` e seus usos |
| **Operadores** | Aritméticos, relacionais, lógicos e bit a bit |
| **Entrada e Saída** | `printf`, `scanf` e formatação de dados |
| **Estruturas de Decisão** | `if`, `else`, `switch` e operador ternário |
| **Estruturas de Repetição** | `for`, `while`, `do-while` |
| **Funções** | Declaração, parâmetros, retorno e recursão |
| **Arrays e Matrizes** | Vetores unidimensionais e bidimensionais |
| **Strings** | Manipulação de cadeias de caracteres |
| **Ponteiros** | Conceito, aritmética de ponteiros e ponteiros para funções |
| **Structs** | Criação e uso de estruturas de dados |
| **Alocação Dinâmica** | `malloc`, `calloc`, `realloc` e `free` |
| **Arquivos** | Leitura e escrita em arquivos com `fopen`, `fclose`, etc. |

Cada tópico contém:
- ✅ **Exemplos comentados** para facilitar o entendimento
- 📝 **Exercícios** para praticar o conteúdo
- 💡 **Dicas** e boas práticas de programação em C

## 🚀 Como usar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/insxnsive/ibmr-c-exercises.git
   cd ibmr-c-exercises
   ```

2. **Navegue até o tópico de interesse** dentro das pastas do projeto.

3. **Compile o exercício** usando o GCC:
   ```bash
   gcc nome_do_arquivo.c -o programa
   ./programa
   ```

> 💻 Você pode usar qualquer compilador C (GCC, Clang, MSVC) ou IDE de sua preferência (Dev-C++, Code::Blocks, VS Code, CLion, etc.).

## 🤝 Como contribuir

Contribuições são muito bem-vindas! Se você é estudante e quer adicionar exercícios, corrigir erros ou melhorar exemplos existentes:

1. Faça um **fork** do repositório
2. Crie uma nova branch: `git checkout -b minha-contribuicao`
3. Faça suas alterações e commit: `git commit -m "feat: adiciona exercício de ponteiros"`
4. Envie para o seu fork: `git push origin minha-contribuicao`
5. Abra um **Pull Request** neste repositório

Por favor, mantenha o código **comentado** e organizado para que outros estudantes possam entender facilmente.
